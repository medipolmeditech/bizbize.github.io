## 🕹️ I. HACKATHONA GELMEDEN ÖNCE KURULMASI GEREKEN ARAÇLAR

### **🔧 1. Git ve GitHub Hesabı**

Hackathon çözümleri GitHub üzerinden teslim edileceği için:

**Git Kurulumu:**
- **Windows:** https://git-scm.com/download/win
- **macOS:** Homebrew → `brew install git`
- **Linux:** `sudo apt install git`

**GitHub Hesabı:**
- Yoksa hemen oluştur: https://github.com

**Temel Komutlar:**
```bash
git clone ...
git add .
git commit -m "mesaj"
git push
git pull
```
